Finance Module:


Following is a list of functionalities of the system. There is a Finance Team user who can login and view list of claims made by employees and approve or reject them. A new finance user can register by providing the proper unique key. the finance user can update his contact details and change the login password.

1. Register a new finance user:

•Insert the values into database table if data is valid, else display appropriate error messages.

•User ID should be provided by the registering user. It should start with FE and contain 5 capital letter characters or numbers or a combination of these

Password should have minimum 5 characters and maximum 50 and must have only numbers or letters or a combination of these.

2. Claims:

•Displays list of claims along with relevant details and provides option to approve or reject them.

3. Upddate details:

•User can change the login password, e-mail and mobile number by clicking on the respective button.
